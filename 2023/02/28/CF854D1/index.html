Codeforces Round 854 D1题解

发表于2023-02-28|更新于2023-02-28|coding changes the world
|字数总计:524|阅读时长:2分钟|阅读量:

题意

有n个程序在等待运行，每个程序的编号是1~k,程序可以运行在CPU1或者CPU2上。

正常情况下程序运行所需时间是cold[i],如果当前这个程序运行的CPU,上一次运行的程序编号和这次一样，消耗的时间更少，为hot[i]

n个程序的序列中,一个程序运行完了才可以运行下一个程序,求最短时间

n,k<=5000

思路

dp[i][j]表示考虑完了前i个程序,上一次和第i个程序运行在不同CPU的程序是第j个,所需要的最短时间

思考之后会发现这个dp需要正着转移,即从i转移到i+1,复杂度才是O(n^2)

根据第i+1个程序运行的CPU与i相同或者不同，dp[i][j]转移到dp[i+1][j]或者dp[i+1][i]
